Replacer

Jul 29th, 2012

Replacer is a simple base class, which is very similar to Mixer. The difference is that it is used to override only existing properties, it doesn’t add new properties to instance.

It can be included with following commands:

Description

Here is its definition:

As you can see it overrides existing properties with non-default values. All other properties are ignored. It is a “tame” version of Mixer, which can be used to keep instances clean of unwanted properties. Replacer can be used as a base, or as a mixin, but usually it serves as the most deepest base to take care of parameters.
