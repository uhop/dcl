dcl.Advice

Jul 29th, 2012

This is a constructor, which serves as a foundation of method decorators used by dcl.advise(). It is not intended to be used by end-users, and useful only for extending dcl itself.

Description

dcl.advise() produces objects based on dcl.Advice, which, in turn, is derived from dcl.Super. It is only useful for extending dcl.

By default a “before” advice (if present) is exposed as a property b on a decorator object, an “after” advice as a property a, and an “around” advice as a property f.
