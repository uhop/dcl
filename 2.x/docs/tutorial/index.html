This file is written using a literate JavaScript. You can download this tutorial to use as a local reference or to run it locally.

If you prefer to see text as inline comments, just click on a sidebar handle at the far right.

While dcl works great in browsers using an AMD loader or even simple <script>, this tutorial is assumed to be run with node.js.

Create a directory, copy this file there, install dcl:

npm install dcl@2

And run the tutorial:

node tutorial.js

Enter AOP

We add before and after advices:

We do not define a constructor for C here – we inherit B’s constructor with the same signature.

before uses the same signature as the advised method.

after always uses the predefined signature.

The first printed number (2) comes from the before advice, the second one (10) comes from the after advice, and the third one (10 again) is from our console.log() printing whatever was returned.

Now let’s add another supercall (an around advice), and add a result modification from an after advice. In reality you should think twice about doing that: an around advice is the more natural place for that. But we do it to demonstrate how it can be done.

If we re-define m() as a regular method, it should behave as a supercall that doesn’t call its super. But existing before and after advices should still continue to function normally.
